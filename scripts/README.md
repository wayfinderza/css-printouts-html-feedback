# Scripts Directory

This directory contains automation tools for the CSS cleanup and PDF generation process.

## Structure

- `converters/` - Tools for HTML to PDF conversion
- `utilities/` - Helper scripts for CSS cleanup and file management

## Available Tools

### CSS Processing
- CSS minification and cleanup scripts
- Removal of animations, transitions, and decorative elements
- Accessibility rule simplification

### HTML Processing
- JavaScript removal and cleanup
- Template generation from source HTML
- Print optimization utilities

### PDF Generation
- HTML to PDF conversion tools
- Batch processing scripts
- Quality control and validation

## Requirements

- Node.js for JavaScript-based tools
- Puppeteer or similar for PDF generation
- CSS parsing libraries for cleanup automation

## Usage

Run scripts from the project root directory. Each tool includes usage documentation.
