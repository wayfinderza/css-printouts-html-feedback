# Samples Directory

This directory contains example projects demonstrating the CSS printout workflow.

## Structure

- `basic-example/` - Simple demonstration of the process
- `github-project-example/` - Real-world example using a GitHub project

## Learning Path

1. Start with `basic-example/` to understand the core concepts
2. Examine `github-project-example/` for a complete workflow
3. Use these as templates for your own projects

## What You'll Find

- Original CSS files from external sources
- Minimal CSS versions focusing on structure
- HTML templates optimized for printing
- Generated PDF examples
- Step-by-step documentation

Each example includes before/after comparisons showing the transformation from complex, styled pages to clean, printable documents suitable for structural feedback.
