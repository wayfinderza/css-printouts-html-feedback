# CSS Files Directory

This directory contains CSS files organized for the printout feedback process.

## Structure

- `external-projects/` - CSS files imported from other GitHub projects
- `minimal-styles/` - Stripped-down, minimal CSS for clear structure focus

## Usage

1. Import CSS files from target GitHub projects into `external-projects/`
2. Create minimal versions in `minimal-styles/` that focus on structure over styling
3. Use these minimal CSS files with HTML pages for cleaner printouts

## Guidelines

- Remove unnecessary animations, transitions, and decorative elements
- Focus on typography, spacing, and layout essentials
- Maintain readability and accessibility
- Keep file organization that mirrors the original project structure
