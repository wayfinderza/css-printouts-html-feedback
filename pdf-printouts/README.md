# PDF Printouts Directory

This directory contains generated PDF files from HTML pages.

## Structure

- `generated/` - Recently generated PDF files
- `archives/` - Historical versions and backups of PDF files

## Usage

1. PDF files are automatically generated from HTML pages in `html-pages/templates/`
2. Files are organized by project name and date
3. Use these PDFs for feedback sessions and reviews

## File Naming Convention

```
[project-name]-[page-name]-[YYYY-MM-DD].pdf
```

Example: `my-webapp-homepage-2024-01-15.pdf`

## Review Process

- Share PDF files with stakeholders for structural feedback
- Focus on content flow, readability, and information hierarchy
- Gather feedback on the raw structure without design distractions
